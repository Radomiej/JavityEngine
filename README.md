# JavityEngine
2D Game Engine for Java, based on LibGdx and Unity concepts

Site: http://radomiej.github.io/JavityEngine

Wiki: https://github.com/Radomiej/JavityEngine/wiki

## Fast installation

Download setup-app: 
Run it.
Select needed dependencies.
Generate project.
Import into your IDE.
